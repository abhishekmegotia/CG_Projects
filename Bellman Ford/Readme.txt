This was an undergrad computer graphics project where the purpose of the project was to show a graphical representation of the bellman ford algorithm.
This project was done between Sept 2014 - Dec 2014.
Softwares required :-
- A  Visual C/C++  compiler  like  Microsoft  Visual  Studio is  required  for  compiling  the  source  code  to  make  the  executable  file  which  can  then  be  directly  executed.
- OpenGL , a software interface for graphics hardware with built  in  graphics  libraries  like  glut  and  glut32, and  header  files  like  glut.h,
